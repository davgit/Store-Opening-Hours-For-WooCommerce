== Description ==

**Store Opening Hours For WooCommerce** is makeing to which time on store will be open and which time on store will be close that all information provide. it will be provide Open Hour time and closing hour time of store


> **Store Opening Hours Features**
>
> * Easy Setup
> * Enable/Disable Option
> * Per Day Can be Setup store multiple time and hour
> * Holiday Can be Setup there
> * Customize Popup Logo, Box text Size and Color
> * As well on Widget you can customize size and color
> * Can be Setup User role wise store time
> * Customize Text of Box and Widget 
> * Mobile friendly
